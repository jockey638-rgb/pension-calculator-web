# 🧮 통합 연금 계산기 (4대 연금)

한국의 **국민연금 + 퇴직연금 + 주택연금 + 기초연금** 4대 연금을 통합 시뮬레이션하는 웹 계산기.
입력값 기반으로 노후 자금 충당률을 자동 진단합니다.

## 🌐 라이브 데모

👉 **[통합연금 계산기 사용하기](https://jockey638-rgb.github.io/pension-calculator-web/)** ← (GitHub Pages 활성화 후 자동 접속 가능)

## 📋 주요 기능

| 연금 | 계산 공식 |
|------|---------|
| 🏛️ 국민연금 | α × (A값 + B값) × (1+0.05n) × MIN(가입년수/20, 1) ÷ 12 |
| 💼 퇴직연금 | 월복리 + PMT 분할 수령 (세전/세후) |
| 🏠 주택연금 | 연령별 종신지급률 (HF 공시 기준) × 주택가격 |
| 🪙 기초연금 | 자격 판정 (소득인정액 ≤ 선정기준) + 국민연금 연계감액 |
| 📊 통합 현금흐름 | 60~95세 연도별 4대 연금 합산 + 물가 인플레이션 반영 |

## ✨ 특징

- **실시간 계산**: 입력 변경 즉시 자동 갱신
- **모바일 반응형**: PC/태블릿/스마트폰 모두 지원
- **인터랙티브 차트**: Chart.js 기반 노후 현금흐름 시각화
- **충당률 게이지**: 시각적 자금 충분도 표시
- **단일 HTML 파일**: 외부 의존성은 Chart.js CDN만

## 🚀 사용 방법

1. 위 [라이브 데모](https://jockey638-rgb.github.io/pension-calculator-web/) 링크 클릭
2. 본인 정보 입력 (생년월일, 가입기간, 소득, 자산 등)
3. 자동으로 4대 연금 월 수령액 + 충당률 계산
4. 60~95세 노후 현금흐름 차트 확인
